## Deep Learning
Deep Learning mimics the network of neurons in a brain. This learning can be supervised, semi-supervised or unsupervised. Deep learning algorithms are constructed with connected layers.
Deep neural networks
- Input Layer(Features)
- Hidden Layer(MLP : Multilayer Perception)
- Output Layer(Prediction)

## Neuron(Perceptron)
- Mathematical function inspired by a biological neuron
- Training : Inputs -> (Forward propagation -> Backpropagate -> Update weights, biases) 반복
- Inference : Inputs -> Forward propagation(Linear Function -> Activation Function) -> Outputs

- Parameters : 컴퓨터가 찾음
    + Weights
    + Biases
- Linear Function : z = (w1x1 + w2x2 + ... + wnxn) + b
- Activation Function : y = f(z)
    + A non-linear function applied by a neuron to introduce non-linear properties in the network
    + By introducing non-linearity we can better capture the patterns in this data
    + Softmax Activation Function
        * Softmax(z_i) = exp(z_i)/Σexp(z_j)
        * Each value ranges between 0 and 1
        * The sum of all values is 1 -> Can be used to model probability distributions
        * Only used in the output layer rather than throughout the network
    + ReLU = max(0, x)

## MLP(Multi Layer Perceptron)
- XOR
- Backpropagation Algorithm

## Training Process
- Update parameters by comparing predictions with ground truth
- Backpropagate difference between predictions and ground truth to modify parameters (especially weights) to make the new prediction more similar to ground truth

## Loss Function
- A method of evaluating "how well your algorithm models your dataset"
- Measure difference between predictions and ground truth
- SE = (y^-y)^2
    + y^ = Predictions
    + y = Ground truth

## Cost Function
- The average of the loss function
- J = 1/2m*Σ(y^-y)^2

## Gradient descent
- The Loss/Cost is used to calculate the gradients
- The gradients are used to update the weights of the neural network
- The gradient descent has a parameter called learning rate(steps)
- If the value of Derivatives is low, take small steps
- If the value of Derivatives is high, take large steps
- Global minimum vs Local minimum

## Training
- Epoch : One epoch is when an entire dataset is passed forward and backward through the neural network only once
- Batch : Since one epoch is too big to feed to the computer at once, we divide it in several smaller batches. So basically a batch size is the number of data point in a batch
- Iterations : The number of batches required to complete one epoch. The model updation will be happening every iteration after every batch in an epoch
    + 3 epoch, 2500 data points, 500 batch size -> 5 iterations per epoch

## Transfer learning
- a model trained on one task is repurposed on a second, related task as an optimization that allows rapid progress when modeling the second task.
- By applying transfer learning to a new task, one can achieve significantly higher performance than training with only a small amount of data.
- Transfer learning is so common that it is rare to train a model for year picture gold natural language processing-related tasks from scratch

## Fine tuning
- Transfer pre-trained parameters to new task
- Freeze + Fine-tune parameters

## Computer Vision
- The science and techonology of machines that see
- Concerned with the theory for building artificial systems that obtain information from images
- The image data can take many forms, such as a video sequence, dpth images, views from multiple cameras, or multi-dimensional data from a medical scanner

## Computer Vision Tasks
- Semantic Segmentation
    + Label each pixel in the image with a category label
    + Don't differentiated instances, only care about pixels
    + No objects, just pixels
- Image Classification
    + Assume given set of discrete labels
    + Single Object
- Object Detection
    + Classification + Localization
        * Image -> Class label, Box in the image (x, y, w, h)
        * Evaluation metric : Accuracy, Intersection over Union
    + Multiple object
- Instance Segmentation
    + Detect instances, give category, label pixels
    + SDS : Simultaneous detection and segmentation
    + Multiple Object

## Convolutional neural network(CNN)
- A type of deep learning neural network commonly used for analyzing visual data, such as images or videos
- Specifically designed to automatically learn and extract relevant features from the input data
- Feature extraction -> Classification
- CNN = (((Conv + ReLU) * N + Max Pooling) * M + (FC + ReLU) * K + Softmax)

1. Convolutional Layer(Filter, Kernel, Weight)
- Input image, activation, feature map -> Output image, activation, feature map
- W1 * H1 * C = Input feature map size
- Hyperparameters : 사람이 결정
    + K = Number of filters
    + F = The filter size
    + S = The stride
    + P = Zero padding
- W2 * H2 * K = Output feature map size
    + W2 = (W1 - F + 2P)/S + 1
    + H2 = (H1 - F + 2P)/S + 1
- Number of parameters = F^2CK weights, K biases
2. Pooling Layer(Subsampling)
- Feature map size reduction, translation invariance
- Max Pooling
3. Dense Layer

## ConvNet
- Input image 32*32*3 -> 6 filters, F=5, S=1, P=0 -> Conv1 28*28*6 -> Max pooling F=2, S=2, P=0 -> Pool1 14*14*6 -> 16 filters, F=5, S=1, P=0 -> Conv2 10*10*16 -> Max pooling F=2, S=2 -> Pool2 5*5*16 -> Dense 400 -> FC3 120 -> FC4 84 -> Softmax
